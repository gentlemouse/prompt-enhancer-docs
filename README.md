## �� 什么是灵犀？

灵犀是一款 Chrome 浏览器扩展，在你按下发送前，用不到一秒升级你的提示词——自动分析意图，匹配最佳策略，让 AI 真正理解你想要什么。

## ✨ 核心能力

- **5 套优化策略自动匹配** — 不是一个模板套所有
  - 短指令 → 补全关键缺失，不画蛇添足
  - 复杂需求 → 构建角色/任务/约束/输出框架
  - 追问修正 → 读懂上下文，无缝衔接
  - 结构完整的提示词 → 只打磨措辞，不动骨架
  - 约束追加 → 精准补充，不破坏原有结构
- **8 种任务场景识别** — 代码、写作、分析、提取、问答、规划、研究、对话
- **5 轮对话记忆** — 追问也能精准润色
- **50+ AI 平台通用** — ChatGPT、Claude、Gemini、DeepSeek、Kimi 等

## 🎯 快速开始

1. 从 Chrome Web Store 安装灵犀
2. 在任意 AI 对话平台输入你的提示词
3. 按 `Cmd/Ctrl+Shift+E` 或点击输入框旁的 ✨ 按钮
4. 一键优化，不到一秒

**免费体验**：10 次优化，开箱即用。接入自己的 API Key，永久无限使用。

## 🔒 隐私承诺

- 不采集任何提示词内容
- API Key 加密本地存储，从不上传
- 匿名统计可一键关闭

详见 [隐私政策](privacy-policy.md)。
